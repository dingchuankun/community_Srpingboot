## 项目简介  
基于SpringBoot+Maven开发，实现了文章列表分页展示，发布文章，Github登陆等功能模块，
前端使用了Bootstrap布局和Thymeleaf模板引擎，数据库使用Mysql,使用Lombok库简化实体类，
项目实现热部署，实现修改立即生效
